# Health App

This is a simple health app landing page.

## Development

To get started, you will need to have [Node.js](https://nodejs.org/en/) installed.

1. Install the dependencies:
   ```bash
   npm install
   ```

2. Run the tests:
   ```bash
   npm test
   ```

This will run the tests using Jest and check if the icons are present in the `index.html` file.
